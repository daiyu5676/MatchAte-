# 🎀 Match It! – Memory Game

A cute, pink-themed memory matching game built using **C++ (via WebAssembly)** and **JavaScript**.

## 🚀 Features
- 🧠 Memory-based card matching gameplay
- 💖 Soft pink themed UI
- 🔥 Streak system with dynamic messages
- ⏱ Timer and 🎯 move counter
- 🏆 Leaderboard with local storage
- ✨ Smooth card flip animations

## 🛠 Tech Stack
- C++ (game logic)
- Emscripten (C++ → WebAssembly)
- JavaScript (UI + interaction)
- HTML/CSS (frontend)
